<h2>Python Syntax, Variable & Data Types</h2>

<h4>Syntax</h4>
<p>
Syntax is the set of rules that define how to write code in a programming language.
</p>

<h4>Variables</h4>
<p>
A variable is a label for storing information. You can reference variables to retrieve or update the stored information.
</p>

<h3>Naming Variables</h3>
<ul>
  <li>Can only contain letters, numbers, and underscores</li>
  <li>Cannot start with a number</li>
  <li>Are case-sensitive</li>
  <li>Use snake_case (words separated by underscores)</li>
</ul>

<h4>Example of Variables</h4>
<pre><code>
name = "daniel"
age = 18
is_number = True

print("My name is", name)
print(age)
print(is_number)
</code></pre>

<h4>Data Types</h4>
<ul>
  <li>Text Type: <b>str</b></li>
  <li>Numeric Type: <b>int, float, complex</b></li>
  <li>Sequence Types: <b>list, tuple, range</b></li>
  <li>Mapping Type: <b>dict</b></li>
  <li>Set Types: <b>set, frozenset</b></li>
  <li>Boolean Type: <b>bool</b></li>
  <li>None Type: <b>NoneType</b></li>
</ul>

<h5>String (str)</h5>
<p>
A string is a sequence of characters wrapped in either single or double quotation marks.
</p>

<h5>Number Types</h5>
<p>
<b>int:</b> A whole number without decimals.<br>
<b>float:</b> A number containing one or more decimal points.<br>
<b>complex:</b> A number with real and imaginary parts.
</p>
