# Python Syntax
<h4>Syntax</h4>
<p>
A variable is a label for storing information you can reference variables to retrieve or update the stored information</p>
<h4>variable</h4>
<p>
Syntax is the set of rules that define how to write code in a programming language
</p>

<h3>Naming Variables</h3>
<ul>
<li>Can only contain letters, numbers and underscores</li>
<li>Cannot start with a number</li>
<li>Are case-sensitive</li>
<li>Use snake case (words are separated by an underscore)</li>
</ul>

<detail>
# Example of variables
name = "daniel"
age = 18
is_number = True

print("My name is", name)
print(age)
print(is_number)
